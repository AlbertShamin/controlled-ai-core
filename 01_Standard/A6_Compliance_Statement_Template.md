**Version:** 1.2

**PUBLIC SUMMARY** – This document is an abbreviated overview of the Controlled AI Core™ Standard. The full implementation specification is not publicly available.

---

## 1. Purpose

Provides the only acceptable format for a formal declaration that a system complies with the standard.

## 2. Key Elements

The statement identifies the system, designates the AISO, declares conformity, and is valid only for a specific configuration; changes invalidate it.

## 3. Use in Audits

Serves as primary evidence, accompanied by logs and test results.

## 4. Relationship

Required by the Audit Framework (A5 – not public).
