## 1. Purpose

Defines requirements for managing AI agents ‚Äď systems capable of autonomous actions and tool invocation.

## 2. Key Principles

- Agents are classified by autonomy level and tool access.
- Each agent must be registered before use.
- Subordinate agents cannot receive more privileges than their coordinator.
- Policies are applied independently to each agent.
