## 1. Purpose

Defines the AI systemŌĆÖs lifecycle phases, with emphasis on controlled decommissioning (Permanent Exit).

## 2. Lifecycle Phases

Design, Development, Deployment, Operation, Transition, Permanent Exit.

## 3. Permanent Exit Requirements

Audit logs must be preserved for a defined period; AISO authorisation is required; system cannot be reactivated after exit.

## 4. Reactivation Prohibition

After Permanent Exit, the system is considered terminated; any restart requires a new lifecycle.
